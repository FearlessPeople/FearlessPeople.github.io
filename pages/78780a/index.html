Markdown学习笔记

# 标题

Markdown的标题使用很简单，直接在文字前方加入#号就代表了标题，一个#号表示一个标题，##二级标题，以此类推。共6级

# 列表

列表分为有序列表和无序列表 无序列表：只需要在文字前方加入-或者*号即可 有序列表：直接使用1. 2. 3.即可

# 粗体和斜体

粗体：用两个*包含的一段文本就是粗体。 斜体：用一个*包含的一段文本就是斜体，或者使用_包裹起来的也是斜体 例如：粗体 斜体

# 代码框

行内代码：需要使用两个`把中间的代码包裹起来。

protected void page_load(){ String str = "abc"; System.out.println(str);}

缩进风格代码块：或者在每行前面打4个空格也同样可以实现代码块

public static void main(){
    System.out.println("Hello Markdown");
}

多语言的代码块：代码块可以使用三个`来定义开始和结束,语言类型也可以省略

public static void main(String[] args){
     System.out.println("Hello Markdown");
}

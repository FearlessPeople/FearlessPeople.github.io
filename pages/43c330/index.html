Intellij IDEA中使用Git

# 1. 将本地项目上传到oschina

# 2. 在oschina的码云中新建项目

# 2.1 然后记住新建项目的http地址

# 3.将本地项目上传到码云

在IDEA中将本地项目合并成一个Git项目 在IDEA中打开你要上传的项目

# 导入项目

菜单栏中：VCS-->Import into Version Control-->Create Git Repository

  1. 在弹出的窗口中，选择你本地项目的根目录即可。
  2. 将项目代码添加进本地仓库中 右键项目-->Git-->Add 注意：该操作不会有任何窗口出现
  3. 将本地项目代码Add到本地仓库 右键项目-->Git-->Commit Directory
  4. 将本地项目代码Commit到本地仓库 在弹出的窗口中点击Commit 注意：当作者不填的时候，好像是会自动生成你电脑的登录名。并且在gitee页面上也不会有头像显示，所以建议填上 当输入名字的时候，会自动识别出用户
  5. 将本地仓库代码推送到服务器（码云）上 菜单栏：VCS-->Git-->Push 接着会弹出一个窗口，在该窗口中填入刚才在码云上新建项目的地址 在弹出的窗口中点击Push即可

# 期间可能出现的问题

# Push rejected: Push to origin/master was rejected

错误分析：意思是git拒绝合并两个不相干的东西 此时你需要在打开Git Bash，然后进入项目的根目录，然后敲git命令：

git pull origin master --allow-unrelated-histories
1

出现类似于这种信息就说明pull成功了：

From https://github.com/danishlyy/HappyBirthday
